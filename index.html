JavaScript Calculator

Instructions:

  1. Open index.html in your google chrome browser.
  2. Press Ctrl + Shift + J (Windows/Linux) or Cmd + Option + J (Mac) to open the console.
  3. Type function calls directly into the console to test:
    • addition(5, 3) → 8
    • subtraction(10, 4) → 6
    • multiplication(2, 6) → 12
    • division(20, 4) → 5
  4. Type history to view the history log.
  5. Try edge cases:
    • division(5, 0) → Cannot divide by zero.
    • addition("s", 2) → Invalid character used.
